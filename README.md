# AI Resume Analyzer

An AI-based web application that analyzes resumes, calculates ATS score, and recommends suitable job roles using NLP and semantic matching.

## Features
- Resume skill extraction
- ATS resume scoring
- Job role recommendation
- Skill gap analysis
- Learning roadmap suggestions

## Tech Stack
- Python
- Streamlit
- Sentence Transformers
- Scikit-learn

## Run Locally
pip install -r requirements.txt  
python -m streamlit run app.py
